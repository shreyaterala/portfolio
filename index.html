Haptic Museum Display

Mechanical Design Lead | Fusion 360, Arduino, Fabrication

Problem: Museums have limited accessibility for those with visual impairments, restricting their engagement with exhibits. Traditional audio guides often fail to convey the physical intricacies of artifacts.

Solution: Developed a tactile display device consisting of a 7x5 pin array with a pin-wise replication control system and a silicone overlay. The tactile display aims to convey shape through the pattern of the actuated pins and color through the height of the corresponding pins.

Engineering Details:

  • Designed custom CAM mechanisms with tight tolerances to ensure pin stability and smooth actuation.
  • Developed the control logic in Arduino to map digital image inputs to pin height outputs.
  • Iterated on the silicone overlay to balance sensitivity with durability.

RUBI - Self Solving Cube

Oct 2024 - Dec 2024 | Mechatronics & CV

Goal: Create a self-solving Rubikâ€™s cube that serves as an interactive learning tool.

Unlike simple solvers that reverse moves, RUBI uses Computer Vision to scan the scrambled state of the cube faces. It then computes an optimal solution algorithm (typically Kociemba or similar) and executes the moves using stepper motors.

Key Components:

  • Vision System: Webcam mounts and OpenCV pipeline to detect face colors.
  • Actuation: Custom 3D printed grippers connected to high-torque stepper motors.
  • Logic: Python control script coordinating vision, solving algo, and motor drivers.

ATL Flight Price Predictor

Machine Learning Project

Project aimed to predict flight costs for Georgia Tech students using Kaggle datasets. I led the model comparison and optimization phase.

Results:

  • Compared KNN, Linear Regression, and Random Forest models.
  • Random Forests achieved the highest accuracy with an R-squared of 0.9557.
  • Visualized feature importance to understand pricing factors (seasonality, airline, duration).
